# Fertility patterns across migrant generations (Finland, cohort 1985-94)

[![R language](https://img.shields.io/badge/language-R-blue)](https://www.r-project.org/)
[![NordForsk funding](https://img.shields.io/badge/funding-NordForsk-green)](https://www.nordforsk.org/projects/network-dynamics-ethnic-integration)
[![Research Council of Finland funding](https://img.shields.io/badge/funding-Research_Council_of_Finland-green)](https://research.fi/en/results/funding/81092)

This repository contains the data and code to replicate the results of the article:
- Estévez, J. L., & Rotkirch, A. (2024). 'Fertility Trends Across Migrant Generations Reexamined: Insights from Finnish Administrative Data'. _SocArXiv_. https://doi.org/10.31235/osf.io/aqw6h

## Purpose of the repository

This repository is dedicated to ensuring transparency and reproducibility in research on the fertility patterns of foreign-background populations in Finland.
It contains all the code used in the analysis presented in the accompanying article.
The data was provided by [Statistics Finland](https://stat.fi/) and can only be accessed with their authorization.
